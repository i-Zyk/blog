HTML学习笔记 

HTML 全称Hyper Text Markup Language;
<meta charset="utf-8">
    charset字符集编码，gb2312国家标准2312条 和gbk国家标准扩展简单，utf-8是unicod（万国码）的升级版；
<html lang="en">
    en：English的简写，zh：中文简写
    告诉搜索引擎和爬虫，我们的网站是关于什么内容的；
<!DOCTYPE html>
    标准盒模型；
    去掉或者写残的话就会变成怪异盒模型；    
<meta content='衣服' name="keyword">
<meta content='这是一件漂亮的衣服' name="description">
    添加keyword和description便于搜索引擎搜索
有序列表
    <ol type='1/a/A/i/I' start='2' reversed='reversed'>
        <li></li>
        <li></li>
        <li></li>
    </ol>
    type是排序5种方法，start='2'从第二个开始排列，reversed倒序排列；
无序列表
    <ul type='disc/square/circle'>
        <li></li>
        <li></li>
        <li></li>
    </ul>
    type='实心圆/方块/空心圆'
img标签
    <img alt='this is fang' title='my name is kk'>
    alt图片占位符，title图片提示符
a标签
    a --> anchor 锚
    href --> HyperText reference
    1.超链接
        <a href='www.baidu.com'>baidu</a>
        <a href='www.baidu.com' target='_blank'>baidu</a>在新标签中打开
    2.锚点
        <a href="#demo">find demo</a>
    3.打电话、发邮件
        <a href='tel:1389231'></a>
        <a href='mailto:12345@qq.com'></a>
    4.协议限定符
        <a href='javascript:void(0)'></a>    
form表单
    <form method='get/post' action='http://213.com'>
        <p>
            username:<input type='text' name='usename'>
        </p>
        <p>
            password:<input type='password' name='password'>
        </p>
        <input type='submit'>
    <form>
input标签
    <input type='radio' name='star' value='1'>
    <input type='radio' name='star' value='2'>
    <input type='radio' name='sex' value='male' checked>
    <input type='radio' name='sex' value='female'>
select标签
    <select name='province'>
        <option value='beijin'>beijing</option>
        <option value='tianjin'>tianjian</option>
        <option value='shanghai'>shanghai</option>
    </select>

vue练习项目流程(二)

–>进入开发
1.搭建项目框架
npm install vue-cli -g
vue init webpack mysell
cd mysell
cnpm install
npm run dev

2.模拟Mock数据 mockjs axios插件安装
cnpm install mockjs –save-dev dev只在开发时候用的
cnpm install axios –save

写好data.json，放在根目录
根目录建mock文件夹，index.js文件
index.js文件中导入mockjs(不是mock.js)和data.json
然后Mock.mock()模拟接口

main引入mock/index,不需要就关掉
  created () {
    axios.get('/good/seller').then(
      res => {
        console.log(res)
        if (res.data.code === 0) {
          console.log(res.data.data)
        }
      }
    )
  }
查看接口是否可用

3.图标字体 https://icomoon.io/
将svg Import Icons –> Generate Font –> Download
将压缩后的 style.css 复制到 mysell/static
fonts 复制到 src/assets 不行的话也复制到 static
在用到图标的style 内导入@import ‘../../static/style.css’;
安装cnpm install css-loader –save-dev

4.页面实现
src/app.vue实现大块 –> components创建模板 –> router/index.js 实现路由跳转
安装cnpm install stylus –> cnpm install stylus-loader
cnpm install postcss-loader css模块化
cnpm install css-loader 解析css文件
cnpm install style-loader 将css文件放入style

将所需要的图片放入assets文件夹，还有常见的基础方法stylus文件夹
将reset.css放入static将原生的一样样式替换掉，初始化的一个文件，有人在维护
--> index.html 中link引入
