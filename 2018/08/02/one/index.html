HTML学习笔记 

HTML 全称Hyper Text Markup Language;
<meta charset="utf-8">
    charset字符集编码，gb2312国家标准2312条 和gbk国家标准扩展简单，utf-8是unicod（万国码）的升级版；
<html lang="en">
    en：English的简写，zh：中文简写
    告诉搜索引擎和爬虫，我们的网站是关于什么内容的；
<!DOCTYPE html>
    标准盒模型；
    去掉或者写残的话就会变成怪异盒模型；    
<meta content='衣服' name="keyword">
<meta content='这是一件漂亮的衣服' name="description">
    添加keyword和description便于搜索引擎搜索
