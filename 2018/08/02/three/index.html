vue练习项目流程(二)

–>进入开发
1.搭建项目框架
npm install vue-cli -g
vue init webpack mysell
cd mysell
cnpm install
npm run dev

2.模拟Mock数据 mockjs axios插件安装
cnpm install mockjs –save-dev dev只在开发时候用的
cnpm install axios –save

写好data.json，放在根目录
根目录建mock文件夹，index.js文件
index.js文件中导入mockjs(不是mock.js)和data.json
然后Mock.mock()模拟接口

main引入mock/index,不需要就关掉
  created () {
    axios.get('/good/seller').then(
      res => {
        console.log(res)
        if (res.data.code === 0) {
          console.log(res.data.data)
        }
      }
    )
  }
查看接口是否可用

3.图标字体 https://icomoon.io/
将svg Import Icons –> Generate Font –> Download
将压缩后的 style.css 复制到 mysell/static
fonts 复制到 src/assets 不行的话也复制到 static
在用到图标的style 内导入@import ‘../../static/style.css’;
安装cnpm install css-loader –save-dev

4.页面实现
src/app.vue实现大块 –> components创建模板 –> router/index.js 实现路由跳转
安装cnpm install stylus –> cnpm install stylus-loader
cnpm install postcss-loader css模块化
cnpm install css-loader 解析css文件
cnpm install style-loader 将css文件放入style

将所需要的图片放入assets文件夹，还有常见的基础方法stylus文件夹
将reset.css放入static将原生的一样样式替换掉，初始化的一个文件，有人在维护
--> index.html 中link引入

app.vue写样式 写之前 UI图的像素要减半
<style lang='stylus' ref='stylesheet/stylus'>
    @import 'assets/stylus/index.styl'
</style>
