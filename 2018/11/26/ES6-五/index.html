ES6(五)

迭代器（遍历器）Iterator 生成器

数据的迭代

  • 数组 对象 Map Set

  • 对于数组的迭代

    1. for
    2. map forEach filter

  • ES6迭代器 是一个对象，专门为可迭代对象设计的统一接口，这个方法用来遍历
    特点： 每个迭代器 都有 next 方法，每次执行next 返回一个结果对象 {value, done}

  1. value 每次迭代的数据，done 迭代是否结束 Boolean
  2. 每个迭代器有专用指针，开始时指向数据结构的第一个值，每次调用next方法，返回当前值，指针下移

  3. 每个迭代器支持 for of 循序

  4. 创建一个可以迭代数组的迭代器

  • 默认的 Iterator 接口

    1. Array Map Set String arguments NodeLists

    2. Array Map Set

      • entries(), keys(), value()

      • 所有迭代器都有 symbol.iterator 属性必须通过中括号形式访问[symbol, iterator]，返回默认的迭代器

      • 原理 数组中 迭代器中的指针，每次指向下一个索引

Map Set 会把邻接链表 转化一个单向链表 指针每次指向一个节点，指针下移

生成器 Generator

  • 用于生成迭代器
  • 生成器 是 函数 * yield

  • yield 没有返回值

  • 委托迭代器 yield *
    遍历数组和字符串
