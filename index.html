什么是css3？

一、css3选择器
    1、属性选择器
        E[att^= 'val'] {} 以val开头
        E[att$= 'val'] {} 以val结尾
        E[att*= 'val'] {} 包含val
        E[att~= 'val'] {} 存在单独demo data=”demo ccc“（可以选到demo） data=”xxdemo“（这个选不了） 
        div[data="demo"] {
            width:100px;
            height:100px;
            background:red
        }
        <div class="demo" data="demo"></div>
    2、初级伪类选择器：伪类用于向某些选择器添加特殊的效果
        root 根标签选择器
            :root {} 和 html {} 效果一样
        :not 否定选择器
            div:not([class="demo"]) {}
            除了class为demo的div以外
    3、empty 空标签选择器
        用来选择没有内容的元素、不在文档树中的元素，这里没有内容指的是一点内容都没有，哪怕是一个空格
    4、:target 目标元素选择器
        用来匹配被location.hash 选中的元素（即锚点元素）
        选择器可用于选取当前活动的目标元素
    5、伪类选择器
       :first-child 第一个子元素
       :last-child 最后一个子元素
       :nth-child (n) {} 第xx个子元素，n代表变量自然数
       :nth-last-child (n) {} 从后往前数
       odd 2n-1奇数 even 2n偶数
        以上四个选择器均有弊端，即如果当前位置元素不是前面所修饰的元素，那么无效

        :first-of-type 第一个子元素
        :last-of-type 最后一个子元素
        :nth-of-type (n) {} 第xx个子元素，n代表变量自然数
        :nth-last-of-type (n) {} 从后往前数
        此种选择器，限制了类型，即在所修饰元素的类型下选择特定位置的元素

        :only-child 唯一子元素选择器
        选择是独生子的子元素，即该子元素不能有兄弟元素，它的父元素只有他一个直接子元素
        注意：选择的元素是独生子子元素，而非有唯一子元素的父元素

        :only-of-type
        如果要选择第某类特定的子元素（p） 在兄弟节点中是此类元素唯一的话 就需要用到这个属性

        :enabled 可用的元素
        :disabled 不可用的元素
        :checked 选择框的被选中状态
         eg：<input type='text' name='age' disabled />

         :read-only 选中只读的元素
          eg：<input type='text' name='age' readonly="readonly" />
         :read-write 选中非只读的元素
          eg：<input type='text' name='age' />

    6、伪元素选择器
        css3对伪元素进行了一定的调整，在以前的基础上增加了一个：
        也就是现在变成了::first-letter,::first-line,::before,::after
        另外还增加了一个::selection

        ::selection
            "::selection"选择器是用来匹配突出显示的文本（用鼠标选择文本的时候）
            浏览器默认情况下，用鼠标选择网页文本是以“蓝色的背景，白色的字体”显示的
            属性：user-select:none
            注：火狐下必须加-moz-
            -moz-::selection
        ::first-letter
            选择p标签内第一个字符
        ::first-line
            选择p标签内文字第一行
        ::before
            标签前面添加样式
        ::after
            标签后面添加样式

    7、条件选择器
        E > F 直接子元素
        E + F 后面的紧挨着的兄弟节点
        E ~ F 后面的兄弟节点

1、border-radius  圆角
    border-radius:20px 40px 60px 80px;
    border-top-left-radius:20px;
    border-top-right-radius:20px;
    border-bottom-left-radius:20px;
    border-top-right-radius:20px 10px;
2、box-shadow  盒子阴影
    box-shadow：x轴偏移量 Y轴偏移量 [阴影模糊半径] 
    [阴影扩展半径] [阴影颜色] [投影方式];
    eg:
        .box_shadow{4px 2px 6px 7px #333 inset;} 
        同一盒子可以加多个阴影，阴影之间可用“，”隔开
3、text-shadow  文本阴影
    text-shadow:X-Offset Y-Offset blur color
    text-shadow:0 1px 1px #fff
4、颜色值RGBA
5、渐变的背景颜色
    css3渐变分为两种
    1）线性渐变（linear-to）
     liner-gradient([direction],color[percent],color[percent],..)
     []内为选填，direction角度的单位为“deg”，也可用to bottom，
     to left，to top left等方式表达
    2）径向渐变（radial-at）
     radial-gradient(shape at position,color[percent],color,...)
     shape:放射的形状，可以为原型circle，可以为椭圆ellipse
     postiion：圆心位置，可以两个值，也可以一个，如果为一个时，
     第二个值默认center即50%。值类型可以为，百分数，距离像素，也可以是方位值
     （left，top...）;/* x轴主半径 y轴次半径 */
6、word-wrap:normal|break-word（强制换行）;
    文字边界换行
7、font-face 引入字体库 http：//www.w3cplus/content/css3-font-face字体库网址
    @font-face{
        font-family:"myFirstFont";
        src:url('Sansation_Light.ttf'),
            url('Sansation_Light.eot')format('eot');
    }
    p{
        font-family:"myFirstFont";
    }
    format:此值指的是你自定义的字体格式，主要用来帮助浏览器识别浏览器对
    @font-face的兼容问题，这里涉及到一个字体format的问题，因为不同的浏览器
    对字体格式支持是不一致的，浏览器自身也无法通过路径后缀来判断字体
    兼容性写法
        @font-face {
            font-famil:'ifont';
            src:url('ifont.eot');//IE9+
            src:url('ifont.eot?#iefix')format('embedded-opentype'),//IE6-IE8
                url('ifont.woff')format('woff'),//chrome、firefox
                url('ifont.ttf')format('truetype'),//chrome、firefox、opera、safari、Android、ios 4.2+
                url('ifont.svg#fontname').format('svg');//ios 4.1-
        }
8.border-image 边框应用背景
    border-image:url(xxx.png) number 切割图片
        stretch 若宽度超出border的宽度的话 拉伸
        repeat 若宽度超出border宽度的话 复制图片
        round （content宽度 / border相应的位置宽度）四舍五入 张图片 填充
        eg：border-image：url('./border.png') 27 round;
9.background-origin 背景图片起始位置
    语法：
    background-origin:border-box|padding-box|content-box;
    参数分别表示背景图片是从边框，还是内边距（默认值），或者是内容区域开始显示
10.background-clip:border-box|padding-box|content-box|no-clip
    默认值border-box

    text：background-clip:text;
    从前景内容的形状（比如文字）作为裁剪区域向外裁剪，如此即可使用背景作为填充色之类的遮罩效果
    注意：webkit独有属性，且必须配合text-fill-color属性
    -webkit-background-clip:text;
    -webkit-text-fill-color:transparent;
    text-fill-color:-webkit-background-clip;
    -webkit-background-clip:text;
11.background-size 背景图片尺寸
    background-size：auto|长度值|百分比|cover|contain
12.background-position 背景图片位置
